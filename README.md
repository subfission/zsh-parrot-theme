# zsh-parrot-theme
Custom theme to imitate Parrot Security prompt for oh-my-zsh

## Installation
Run these commands to install the latest version and enable.

    git clone https://github.com/subfission/zsh-parrot-theme.git --depth=1
    cd zsh-parrot-theme
    zsh install.zsh
